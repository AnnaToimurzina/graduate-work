Трекер задач (Task Tracker) представляет собой инструмент, который помогает организовывать, отслеживать и управлять задачами в проекте или команде.

Для запуска проекта выполните следующие шаги:

Склонируйте репозиторий с помощью команды git clone https://github.com/AnnaToimurzina/graduate-work.git.

Установите необходимые зависимости, выполнив команду pip3 install -r requirements.txt.

Примените миграции для настройки базы данных: python3 manage.py migrate.

Запустите сервер командой python3 manage.py runserver.

Создайте суперпользователя, используя команду python3 manage.py createsuperuser.

Получите токен аутентификации через эндпоинт /api/token/.

Исследуйте и взаимодействуйте с API, используя полученный токен для аутентификации.
